## Finding Your Store Number

1. Visit https://www.wegmans.com/api/stores
2. Search for your city or address
3. Note the `storeNumber` field
4. Update `config/settings.py`: `ALGOLIA_STORE_NUMBER = YOUR_NUMBER`

## Store Data Fields

Each store in the API includes:
- `storeNumber` - Unique identifier (critical for Algolia API)
- `streetAddress`, `city`, `stateAbbreviation`, `zip`
- `latitude`, `longitude` - GPS coordinates
- `phoneNumber` - Store contact
- `hasPickup`, `hasDelivery`, `hasPharmacy`, `hasECommerce`
- `aislePositionMapping` - Complete aisle layout (JSON string)
- `alcoholTypesForSale` - Wine, Beer, Spirits availability
- `openingDate`, `ecommerceOpeningDate`
- `zones` - Regional grouping

## Technical Notes

- API endpoint: `https://www.wegmans.com/api/stores`
- Returns 114 stores (some are liquor-only stores without food)
- Store numbers are NOT sequential (many gaps)
- Each store has unique aisle layout (`aislePositionMapping`)
- Algolia API requires exact `storeNumber` for correct results
